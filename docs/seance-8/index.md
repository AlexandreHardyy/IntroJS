# Séance 8 : Projet de Validation

## Objectif
Cette dernière séance est consacrée à la réalisation et à la finalisation d'un mini-projet complet mettant en œuvre l'ensemble des concepts vus durant le cours.

## Idées de Projets
1.  **To-Do List avancée** : Ajout, suppression, filtres (tous/faits/à faire), sauvegarde localStorage.
2.  **Application Météo** : Utilisation d'une API Météo (ex: OpenWeatherMap), recherche par ville, affichage dynamique.
3.  **Quiz interactif** : Questions, gestion du score, timer.

## Critères d'Évaluation
*   **Fonctionnalités** : L'application fonctionne-t-elle comme attendu ?
*   **Qualité du Code** : Variable bien nommées, indentation, utilisation de fonctions.
*   **Interface (DOM/CSS)** : L'application est-elle utilisable et un minimum stylisée ?
*   **Concepts Clés** : Utilisation correcte des événements, de la manipulation du DOM, et éventuellement de l'asynchronisme.

## Déploiement
Petite démo de comment mettre son projet en ligne (ex: GitHub Pages, Vercel, Netlify).
