# Ressources Utiles

Voici une liste de liens pour approfondir vos connaissances ou trouver de l'aide.

## Documentation Officielle
*   [MDN Web Docs (Mozilla)](https://developer.mozilla.org/fr/docs/Web/JavaScript) : La bible du développeur web. Toujours consulter en premier.

## Apprentissage Interactif
*   [FreeCodeCamp](https://www.freecodecamp.org/) : Exercices pratiques gratuits.
*   [Codeademy](https://www.codecademy.com/) : Cours interactifs (certains payants).

## Outils & Environnement
*   [VS Code](https://code.visualstudio.com/) : L'éditeur de code recommandé.
*   [Node.js](https://nodejs.org/) : Pour exécuter JS hors du navigateur.

## Exercices d'Algorithmique
*   [Codewars](https://www.codewars.com/) : Katas (défis) de code.
*   [LeetCode](https://leetcode.com/) : Plus orienté entretiens techniques.
