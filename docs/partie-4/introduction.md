# Introduction à NPM et Vite

## Qu'est-ce que NPM ?

NPM (**Node Package Manager**) est le gestionnaire de paquets par défaut pour Node.js. 
Il permet d'installer, de partager et de gérer les dépendances (les bibliothèques de code créées par d'autres développeurs) au sein de vos projets JavaScript.

Plutôt que de réinventer la roue ou de télécharger manuellement des fichiers de code, NPM vous donne accès à un registre immense de packages prêts à l'emploi.

## Et Vite ?

Vite (prononcé "vit", comme le mot français) est un outil de construction (*build tool*) moderne et extrêmement rapide pour le développement web.
Il a été conçu pour résoudre les problèmes de lenteur rencontrés avec d'anciens outils comme Webpack.

**Pourquoi l'utiliser ?**
- Un démarrage du serveur de développement quasi instantané.
- Un rechargement à chaud (Hot Module Replacement) ultra-rapide lorsque vous modifiez un fichier.
- Une configuration minimale (souvent prête à l'emploi).

Dans les prochains chapitres, nous verrons comment initialiser un projet, installer des paquets et comprendre l'écosystème NPM.
